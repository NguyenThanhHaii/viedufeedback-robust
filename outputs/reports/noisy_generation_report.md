# Noisy Test Generation Report

## 1. Mục tiêu

Giai đoạn này tạo các phiên bản test set bị làm nhiễu để phục vụ đánh giá độ bền của mô hình trên tiếng Việt phi chuẩn.

Chỉ test set được làm nhiễu. Train và validation giữ nguyên.

## 2. Noise variants

- `no_accent`: bỏ dấu tiếng Việt.
- `typo_light`: lỗi gõ nhẹ.
- `typo_medium`: lỗi gõ mức trung bình.
- `teencode_light`: thay một số từ/cụm bằng viết tắt phổ biến.
- `mixed_light`: kết hợp teencode, bỏ dấu và typo nhẹ.

## 5. Notes

- Các nhãn sentiment/topic được giữ nguyên từ clean test set.

- `test_eval_all.csv` chứa cả clean và noisy variants để phục vụ evaluation sau này.
- Stage này chưa đánh giá model.
